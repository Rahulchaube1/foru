# CodeBook

**AI-Powered Code Understanding Platform**

CodeBook is a comprehensive platform that combines AI-powered code documentation (like Google CodeWiki), full repository management (like GitHub), and unlimited cloud storage â€” all in one place.

## Features

### AI-Powered Documentation
- Auto-generated wiki pages that update with every commit
- Interactive knowledge graphs and architecture diagrams
- AI chat assistant with full codebase context
- Symbol linking and cross-reference navigation

### Repository Management
- Create, fork, and manage repositories
- Full code browser with file tree and syntax highlighting
- Issues tracker with labels, assignees, and comments
- Pull requests with diff viewer and code review
- Stars, forks, and social features
- Contribution graph and activity feed

### Cloud Storage
- **Unlimited** cloud storage for all files and assets
- Drag-and-drop file upload
- Folder organization and file sharing
- Support for all file types

### Code Snippets
- Create and share code snippets (like GitHub Gist)
- Syntax highlighting for 100+ languages
- Public and private snippets
- One-click copy

### Additional Features
- Beautiful dark theme with modern UI
- User profiles with achievements
- Explore page with trending repositories
- Full settings and customization
- Responsive design for all devices

## Tech Stack

- **React** - UI framework
- **Vite** - Build tool
- **Tailwind CSS** - Styling
- **React Router** - Client-side routing
- **Lucide React** - Icons

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
